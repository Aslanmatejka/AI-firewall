## 30-second version

1. Install Python 3.10+ and run `pip install -r python\requirements.txt`
2. Run `.\scripts\start.ps1`
3. Open **http://127.0.0.1:9470** in your browser
4. When AI apps ask for access, click **Allow** or **Deny** on the dashboard

That is the whole product in four steps. Everything else in these docs is detail.
